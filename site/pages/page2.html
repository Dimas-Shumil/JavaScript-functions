2. Функции без параметров

Что такое функция

Вспомним аналогию с магазином мобильных телефонов. Продавец, скорее всего, не носит с собой калькулятор, чтобы постоянно подсчитывать налоги на товар и финальную стоимость всех покупок. Поэтому в каждом магазине есть кассовые аппараты, которые выполняют эти функции за продавцов.

Так и в вашей программе будут повторяющиеся куски кода, которые возвращают какой-то результат. Функции дают названия секциям кода, которые будет удобно вызвать по имени.

Функция — это блок программного кода, который определяется один раз и может выполняться (вызываться) многократно.

Функция в JavaScript — это объект, который может быть вызван как подпрограмма.

ВАЖНО! Все функции в JavaScript — это объекты!

В дополнение к своим свойствам функция содержит исполняемый код и определённое состояние, которые определяют поведение функции при вызове.

JavaScript имеет множество встроенных функций, например console.log, alert, prompt, с которыми вы познакомились в предыдущем модуле:

console.log('hello');

Чтобы объявить свою функцию, используется ключевое слово function:

function funcName() {

const str = 'My first function';
console.log(str);

}

После function идёт имя функции funcName, круглые скобки и тело функции в фигурных скобках { // code }.

Чтобы вызвать функцию, надо использовать её имя и круглые скобки:

funcName()

В прошлом модуле в разделе «Циклы» вы выполняли следующее задание:

Дан массив. Нужно вывести в консоль количество чётных и нечётных элементов в массиве. Если в массиве есть нулевой элемент, то он учитывается и выводится отдельно.

При выполнении задания необходимо учесть, что массив может содержать не только числа, но и, например, знаки, null и так далее.

На этот раз оформите решение в виде функции, постарайтесь дать этой функции корректное название, вызовите функцию, проанализируйте синтаксис.
