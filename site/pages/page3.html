3. Function declaration vs Function expression

Два способа объявления функции

Классическое объявление функции называется Function Declaration (FD).

Пример:

function sum() {

const result = 1+2;
console.log(result);

}

Другой подход к объявлению функций — Function Expression (FE).

Пример:

const sum = function() {

const result = 1 + 2;
console.log(result);

}

Отличия:

· Функции, объявленные как Function Declaration, создаются интерпретатором до выполнения кода, поэтому их можно вызвать до объявления.
· Функции, объявленные как FE, ведут себя по-разному в зависимости от того, какой оператор был использован. Функции, объявленные как FD, всегда ведут себя одинаково.

func1(); // ошибки нет

func2(); // ошибка

function func1(a, b) {

console.log('function 1 - FD');

}

const func2 = function() {

console.log('function 2 - FE');

};

func1(); // ошибки нет

func2(); // ошибки нет
