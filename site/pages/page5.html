5. Функции высшего порядка

Функциональное программирование

Понятие функции одно из основных в программировании. С помощью них можно реализовать всё что угодно, делая код переиспользуемым и лаконичным. С каждым днём всё популярнее становится функциональное программирование, основой которого, как можно догадаться, является функция.

Особенностью функционального подхода является его декларативность — когда мы описываем конечный результат, а не полный алгоритм действия.

Например, императивный подход на вопрос «Как пройти в библиотеку?» отвечает:

  1. Садись на автобус.
  2. Выходи на остановке «улица Ленина».
  3. Иди прямо.
  4. Поверни направо.
  5. Найди здание с номером 52.
  6. Открой дверь.

Декларативный же подход отвечает на тот же самый вопрос гораздо короче — «Ленина, 52».

Но это не единственная особенность. Также с понятием функционального программирования тесно связаны чистые функции, иммутабельность данных и многое другое, о чём мы будем говорить позднее.

А сейчас разберём определение first-class citizen or object — функции высшего порядка (или функции первого класса, или объекты первого класса).

В JavaScript все функции являются объектами первого класса — они могут быть переданы как аргумент, возвращены из функции и присвоены переменной.

Пример:

function example(func){

// вызываем функцию, которую приняли как аргумент

func();

// возвращаем функцию

return function(){

console.log('internal function')
}

}

// функция, которую будем передавать как аргумент в функцию example

function argFunc(){

console.log('function from arguments')

}

// в переменную resultFunc записываем результат выполнения функции example

// функция, которая передаётся как аргумент, должна быть без круглых скобок

const resultFunc = example(argFunc);

// после вызова example() внутри неё будет вызвана функция argFunc, которую мы передали как аргумент, и в консоли будет текст: function from arguments

// вызываем функцию

resultFunc(); // 'internal function
